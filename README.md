# Food Now 🌿

A responsive, elegant, and modern single-page restaurant landing website built with Bootstrap 5. **Food Now** showcases a vibrant, healthy vegan menu accompanied by an intuitive, user-friendly interactive order form.

🚀 **[View the Live Demo here!](https://dhruvi0515.github.io/Bootstrap/)**

## ✨ Features

* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices using Bootstrap's mobile-first fluid grid system.
* **Immersive Hero Carousel:** An automated, high-resolution slide banner showcasing signature dishes with custom text overlays.
* **Curated Menu Grid:** A visually appealing card layout featuring specific dishes, detailed ingredients, and premium image placements.
* **Streamlined Order Form:** A clean input form equipped with text elements, telephone verification fields, custom option dropdown selects, and instructions text areas.
* **Cohesive Aesthetic:** Features a custom `#4e9c79` sage-green organic theme color running across the navigation bar and footer.

---

## 🛠️ Technologies Used

* **HTML5:** Semantics and modern web structuring.
* **CSS3:** Inline style adaptations for theme matching.
* **Bootstrap v5.3.0:** Utilized for rapid responsive component layouts (`navbar`, `carousel`, `cards`, `forms`, grid system).
* **Unsplash & iStock Images:** Used for high-quality, lightweight placeholder imagery.
